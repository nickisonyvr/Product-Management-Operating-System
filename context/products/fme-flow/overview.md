# FME Flow

> Product context for **FME Flow**. Edit sections as your understanding evolves.

## Product Overview

FME Flow is the server-side platform of the FME Platform that automates, schedules, and deploys the data integration workflows built in FME Form. It enables enterprise teams to run workflows on a schedule, in response to events, or in real time as data arrives — without manual intervention. FME Flow supports deployment as a Safe-managed SaaS (FME Flow Hosted), on-premises, or via AWS and Snowflake Marketplaces.

**Core Value Proposition**: FME Flow automates the data integration workspaces you build in FME Form, running them at scale on a schedule, in response to events, or in real time — without manual intervention.

**Tagline**: *Automate your data workflows at scale — on schedule, on demand, or in real time.*

## Strategy / Positioning

FME Flow sits at the intersection of data integration and enterprise automation, capitalizing on the growing need for real-time, event-driven data pipelines across industries with complex geospatial and operational data needs. As AI adoption accelerates, FME Flow is positioning as the orchestration layer — an "All-Data, Any-AI Platform" — enabling customers to integrate any data source and orchestrate any AI model. It has been recognized as a Gartner Magic Quadrant Niche Player for six consecutive years and a G2 Leader across 7 grids in Spring 2026.

## Key Capabilities

### 1. Automations
Instant, event-triggered workflows that keep data synchronized across connected systems without manual intervention.

### 2. Real-time data streams
Continuous monitoring and processing of live data for instant insights and alerts.

### 3. No-code web apps
Self-serve data products for enterprise users, built without infrastructure coding.

### 4. Data virtualization
Access and expose data via OpenAPI interfaces without physical replication or complex infrastructure.

### 5. MCP server builder
Build and host MCP servers on top of existing FME workspaces, exposing them as tools consumable by AI agents and LLMs.

### 6. Multiple deployment options
FME Flow Hosted (Safe-managed SaaS on AWS), on-premises (Windows or Linux), and marketplace integrations (AWS, Snowflake). On-premises deployments additionally support distributed deployment, dynamic engine licensing, Active Directory, VPN/virtual network integration, full file system access, and the complete format library including Esri Geodatabase (ArcSDE).

### 7. Full format support
On-premises deployments support the complete FME format library, including Esri Geodatabase (ArcSDE), File Geodatabase, and formats requiring third-party Windows libraries. (FME Flow Hosted has Linux-based format limitations — see FME Flow Hosted docs for the formats matrix.)
