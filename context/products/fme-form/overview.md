# FME Form

> Product context for **FME Form**. Edit sections as your understanding evolves.

## Product Overview

FME Form is the desktop authoring environment in the FME Platform for building data integration workflows (called "workspaces") without writing code. Using 800+ built-in transformers, users connect any data source, transform it to their requirements, and output it wherever it needs to go — then optionally publish to FME Flow for automation and scheduling.

**Core Value Proposition**: FME Form is the no-code desktop tool for building data integration workflows that connect any source, transform data to your needs, and output it anywhere — ready to run manually or publish to FME Flow for automation.

**Tagline**: *Connect, transform, and deliver any data — no code required.*

## Strategy / Positioning

FME Form serves as the primary entry point to the FME Platform, giving data professionals a no-code alternative to hand-coded ETL pipelines. Its breadth of connectivity across thousands of formats and systems — and depth of transformation capability via 800+ transformers — makes it the authoring tool of choice for organizations with complex, multi-format data needs, particularly in geospatial, utilities, government, and AEC sectors. Workspaces built in FME Form are designed to be published directly to FME Flow for automation and enterprise-scale deployment.

## Unique Approach

| Pillar | Description |
|--------|-------------|
| **Visual workflow authoring** | Drag-and-drop canvas with 800+ ready-made transformers for building complex data integration logic without code |
| **Deep inspection tools** | Breakpoints, caching, and real-time data visualization to examine and debug data mid-workflow without rerunning from scratch |
| **AI Assist** | Real-time authoring suggestions for building expressions, SQL queries, regular expressions, and documentation |
| **FME Hub** | Community and Safe-built library of pre-built transformation components, extending core capabilities |

## Key Capabilities

### 1. 800+ transformers
Ready-made processing components covering data manipulation, filtering, conversion, and validation across thousands of formats and systems.

### 2. AI Assist
Real-time suggestions for building expressions, SQL, regex, and documentation to accelerate workspace authoring.

### 3. Workspace Compare & Merge
Version control for workflow definitions, allowing teams to identify differences and merge changes across workspace versions.

### 4. Breakpoints & inspection
Pause mid-process to examine data at any point in the workflow without rerunning from the start.

### 5. FME Hub integration
Access community and Safe-built transformation components directly from the authoring environment.
