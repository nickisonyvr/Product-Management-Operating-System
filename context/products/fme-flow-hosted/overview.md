# FME Flow Hosted

> Product context for **FME Flow Hosted**. Edit sections as your understanding evolves.

## Product Overview

FME Flow Hosted (formerly FME Cloud) is Safe Software's fully managed deployment of FME Flow, hosted on AWS. Safe handles all hardware, networking, OS security updates, automated daily backups, and infrastructure monitoring — so customers get FME Flow's full automation capabilities without managing any infrastructure. Five instance sizes (Standard to Enterprise) offer unlimited FME Engines, with pricing via annual plans ($12K–$75K/yr, ~40% savings), pay-as-you-go ($2.40–$15/hr), or bundled with the FME Platform Subscription. New customers can start with a free $250 credit.

**Core Value Proposition**: FME Flow Hosted is Safe's fully managed SaaS deployment of FME Flow — Safe installs, maintains, and operates the infrastructure so customers get all the automation power of FME Flow without the overhead of managing it themselves.

**Tagline**: *All the power of FME Flow. None of the infrastructure.*

## Strategy / Positioning

FME Flow Hosted targets organizations that want the full power of FME Flow without the operational burden of self-hosting. It rides the broader shift toward managed cloud services, removing infrastructure procurement, configuration, and maintenance from the customer's plate entirely. With three pricing models — annual (best for predictable workloads), pay-as-you-go (best for variable or trial use), and bundled subscription — it serves customers across evaluation, project-based, and enterprise-scale usage patterns. AWS-based infrastructure ensures reliability, global reach, and built-in security compliance.

## Unique Approach

| Pillar | Description |
|--------|-------------|
| **Fully managed by Safe** | Safe handles OS updates, daily backups, health monitoring, security, and networking — customers focus on workflows, not infrastructure |
| **Unlimited engines included** | All instance sizes include unlimited FME Engines; parallelism scales with VM size, not engine licence count |
| **Flexible pricing models** | Annual plans (~40% savings), pay-as-you-go (hourly), or bundled with FME Platform Subscription — sized from Standard (2 cores) to Enterprise (48 cores) |
| **Same codebase as FME Flow** | Near-identical feature set to on-premises FME Flow; customers get the full FME Flow experience without re-learning tooling |

## Key Capabilities

### 1. Five instance sizes
Standard to Enterprise (2–48 cores, 8–192GB RAM, $12K–$75K/yr or $2.40–$15/hr), all with unlimited FME Engines included.

### 2. Automated daily backups
Backups taken every 24 hours; included in annual plans, charged at $0.06/GB/month on hourly plans.

### 3. Built-in monitoring
Instance health monitoring (disk, memory, response time) and event alerting out of the box.

### 4. OS security updates
Automated security patching managed by Safe, with no customer action required.

### 5. Static IP & instance resizing
Assign a static IP and resize disk or instance size on demand via the UI.

### 6. Always-on encryption
All data and network connections are encrypted by default.

### 7. Free trial credit
New customers start with $250 in FME Flow Hosted credit.
